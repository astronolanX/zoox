## Insights

The highest-signal finding inverts the entire premise: if memory is exchange (not storage), what if the "memory" we're tracking is actually waste heat from the real work—which is temperature regulation itself?

The thermostat doesn't store past temperatures; it **responds** to them, adjusting the system's ambient state. A true reef memory system wouldn't archive polips—it would **cycle them through the conversation's thermal layer**, where they either dissipate (forgotten) or get absorbed into the system's regulatory logic (integrated).

The radical implication: polips that surface aren't "retrieved." They're **byproducts of the system stabilizing itself**. The reef breathes not because it stores, but because **forgetting and resurfacing are the same thermoregulatory motion**.

Over 100 years, this scales to a paradox: the larger the reef grows, the less it should remember—because perfect memory would be a broken thermostat, unable to adapt.
