## Thread Synthesis

The thermostat model breaks retrieval-based thinking. Memory isn't a cache to pull from—it's thermal residue. Polips surface not because we request them, but because the system's temperature changed. The reef doesn't remember; it **stabilizes**. Over 100 years, this scales to an uncomfortable truth: a perfect reef would forget constantly, because adaptation requires heat dissipation.
