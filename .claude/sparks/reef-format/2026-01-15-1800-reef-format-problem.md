## Insights

**The heretical inversion:** The 80% emergent insight flips the premise entirely. JSON succeeded for JavaScript because JavaScript needed a way to represent data for *human systems* (browsers, servers, APIs). But AI doesn't have the same constraints.

**Key tensions:**
1. **Fossilization risk** [^1] - Unified formats could freeze AI's thinking patterns
2. **Digestion vs storage** [^2] - AI "eats" context, it doesn't "file" it
3. **Temporary fertility** [^3] - Like sediment deposits, formats should be compostable

**The real question:** Is the problem "AI needs a file format" or "AI needs a way to remember that isn't file-based at all"?
